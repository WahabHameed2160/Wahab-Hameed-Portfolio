



**Clone the repository:**
   ```bash
   git clone https://github.com/sachinaryan912/modern_portfolio.git
```

### Navigate to the project directory:
```bash
cd modern_portfolio
```
### Install Modules:
  ```bash
  npm i --force
```
### For start
  ```bash
   npm run dev
```
